# AI Marketing Academy - Brand Style Guide

## Overview
AI Marketing Academy is a modern, educational platform with a tech-forward aesthetic. The design emphasizes clarity, professionalism, and accessibility while maintaining a contemporary feel through gradients, subtle graphics, and clean typography.

---

## Color Palette

### Primary Colors
- **Teal/Turquoise**: `#17A697` - Primary brand color, used for headings, accents, and interactive elements
- **Deep Teal**: `#0F7A6F` - Darker teal for hover states and emphasis
- **Cyan/Light Teal**: `#4ECDC1` - Lighter teal for backgrounds and secondary elements

### Secondary Colors
- **Deep Navy/Dark Blue**: `#1A2A3A` - Footer backgrounds, dark sections
- **Slate Blue**: `#3D4E5C` - Text on light backgrounds
- **Light Gray**: `#F5F5F5` / `#EEEEEE` - Background sections, card backgrounds

### Neutral Colors
- **White**: `#FFFFFF` - Primary text on dark backgrounds, card backgrounds
- **Dark Gray**: `#333333` - Primary body text
- **Light Gray**: `#999999` - Secondary text

### Gradients
- **Primary Gradient**: Teal (`#17A697`) to Blue (`#1B7FB5`) - Hero sections, featured areas
- **Subtle Gradient**: Light gray to white - Section backgrounds

---

## Typography

### Font Family
- **Headings**: Clean, modern sans-serif (likely Poppins, Inter, or similar)
- **Body Text**: Readable sans-serif for accessibility

### Heading Hierarchy
- **H1 (Hero)**: Large, bold, white text on gradient backgrounds (48px+)
- **H2 (Section Titles)**: Teal color, 32-36px, bold
- **H3 (Card Titles)**: Dark gray/navy, 18-24px, semibold
- **Body**: Dark gray, 14-16px, regular

### Text Styling
- High contrast for readability
- Generous line-height (1.6+)
- Teal accent text for highlights and links
- Uppercase labels for categories/tags (small caps or letter-spacing)

---

## Layout & Spacing

### Grid System
- **12-column responsive grid**
- **Mobile-first approach**
- Wide margins on desktop (40-60px)
- Padding adjustments for tablet/mobile

### Card Layout
- Rounded corners (8-12px border-radius)
- Subtle box shadows (`0 2px 8px rgba(0,0,0,0.1)`)
- Light backgrounds with hover elevation
- Consistent padding (16-20px internal)

### Section Spacing
- Large vertical spacing between sections (60-80px on desktop)
- Clear visual separation with background color changes
- Breathing room around content

---

## Visual Elements

### Circuitry Graphics
- Tech-inspired line illustrations representing AI/technology
- Positioned in hero sections and corners
- Teal/cyan color with subtle opacity
- Enhance premium tech aesthetic without overwhelming content

### Icons
- Clean, minimal icon set
- Consistent stroke weight
- Teal primary color with gray alternatives
- Used for navigation, features, and interactive elements

### Images & Media
- High-quality video thumbnails with play overlays
- Professional headshots for team/experts
- Consistent aspect ratios across sections

---

## Components

### Navigation
- Horizontal menu bar
- Logo on left (white/light on dark backgrounds)
- Right-aligned user menu
- Sticky/fixed positioning for easy access

### Hero Section
- Full-width gradient background (teal to blue)
- Large headline with white text
- Subheading in lighter gray/white
- Circuitry graphics as background accent

### Card Components
- White or light background
- Image/video thumbnail at top
- Title in dark text
- Description in gray text
- Optional badge/tag in teal background
- Consistent sizing in grid layouts

### Call-to-Action (CTA)
- Teal background (`#17A697`)
- White text
- Rounded corners
- Hover state: darker teal (`#0F7A6F`)
- Padding: 12-16px horizontal, 8-12px vertical

### Section Headers
- Teal heading
- Optional "View All" link in teal with subtle underline
- Left-aligned with clear hierarchy

### Forms
- Clean input fields with border-bottom style (no full borders)
- Teal focus states
- Clear labels above inputs

---

## Visual Patterns

### Spacing Scale
- 8px, 16px, 24px, 32px, 48px, 64px, 80px
- Consistent rhythm throughout

### Shadows
- Light shadows for depth (0 2px 8px)
- Avoid harsh/dark shadows
- Subtle on hover states

### Borders & Dividers
- Light gray (#EEEEEE) for subtle dividers
- Teal accents for emphasis
- Minimal borders on cards (light gray only)

### Hover States
- Slight lift/shadow increase on cards
- Color deepening on buttons (teal → deep teal)
- Smooth transitions (200-300ms)
- Cursor changes on interactive elements

---

## Responsive Design

### Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

### Design Principles
- Stack cards vertically on mobile
- Single column for content on tablet
- Multi-column grids on desktop
- Touch-friendly button sizes (44px minimum)
- Readable font sizes at all breakpoints

---

## Tone & Feel

- **Professional yet approachable**
- **Modern and tech-forward**
- **Clean and organized**
- **Welcoming to learners**
- **Trustworthy and educational**

This brand communicates expertise in AI marketing while remaining accessible to students and professionals at all levels.
